# ReportConversionProgram (Good Morning)

Программа для конвертации исходных Excel-таблиц и автоматического составления новых отчетов на основе полученных данных.

## Основные возможности
- **Формирование отчетов:** Генерация и запись данных в различные листы итогового Excel-файла:
  - **ЖП** (JP)
  - **СЗ** (CZ)
  - **Бам по УП** (BAM)
  - **Общая информация** (Обобщение данных)
  - **Отслеживание**
- **Графический интерфейс (GUI):** Полностью переработано, тепрерь это приложение с HTML/CSS и java, что позволило многократно улучшить внешний вид приложения.
- **Системные уведомления:** Оповещение пользователя о завершении работы через всплывающие уведомления Windows (библиотека `plyer`).
- **Умное обновление:** При запуске в первые 5 дней текущего месяца программа автоматически предлагает обновить данные за прошлый месяц.
- **Режимы работы:** Поддержка как интерактивного режима с GUI, так и фонового (headless) запуска через консоль.

## Требования и установка
Для работы программы необходим **Python 3.x**. 

Основные используемые библиотеки:
- `tkinter` (обычно встроена в Python)
- `plyer` (для системных уведомлений)
-   Библиотеки для работы с Excel (например, `openpyxl` или `pandas` — в зависимости от реализации модулей в папке `scripts/`).

    
Установить внешние зависимости можно через pip:
```bash
pip install plyer openpyxl
```

## Использование

### Запуск с графическим интерфейсом
Запустите основной файл `Body.py`:
```bash
python Body.py
```

### Запуск без GUI (Консольный / Фоновый режим)
Для запуска обработки без открытия графического окна используйте флаг `--nog`:
```bash
python report_conversion.py --nog
```

### Аргументы командной строки
При запуске через `report_conversion.py` доступны следующие параметры:
- `--console` — запустить программу с отображением окна консоли.
- `--cfile` — открыть директорию с конфигурационным файлом.
- `--pstart` — моментальный старт обработки данных.
- `--nog` — запустить без графической оболочки (headless-режим).

## Сборка в EXE-файл (Windows)
Для создания автономного исполняемого файла используется `pyinstaller`. 
Команда для сборки (выполнять из корня репозитория):

```bash
pyinstaller --onefile --icon=static/icons/new_icon_report-conversion-program.ico -w --hidden-import plyer.platforms.win.notification --name="Good_Morning" --add-data "static;static" Body.py
```
*(Примечание: разделитель `;` в `--add-data` используется в Windows. Для сборки под Linux/macOS замените его на `:`)*

## Структура проекта
- **`Body.py`** — Точка входа, отвечающая за отрисовку GUI (`MainGUI`) и инициализацию бизнес-логики.
- **`report_conversion.py`** — Ядро программы (`ReportConversion`). Содержит всю логику обработки данных, управления конфигурацией и запуска подпрограмм.
- **`programm/`** — Директория с модулями конкретных отчетов (`program_jp`, `program_bam`, `program_cz`, `program_generalization`, `program_for_tracking`).
- **`scripts/`** — Вспомогательные утилиты (запись в Excel `excel_enter`, работа с JSON-конфигами, логирование).
- **`static/`** — Статические ресурсы (иконки, конфигурационные файлы и т.д.).
