# Report Convention Program (GoodMorning)

## Описание
Программа для конвертации исходных Excel таблиц и составления на основе полученных данных нового отчета.

## Назначение
Этот проект предназначен для автоматизации обработки Excel-файлов, включая:
- Конвертацию исходных Excel-таблиц
- Обработку и анализ данных
- Генерацию новых отчетов на основе полученных данных

## Структура проекта
- `BamProgram.py` - модуль для обработки BAM данных
- `Body.py` - основная логика программы
- `Config.py` - файл конфигурации
- `CzProgram.py` - модуль для обработки CZ данных
- `ExcelPrint.py` - модуль для работы с Excel
- `GeneralizationProg.py` - модуль общих функций
- `JpProgram.py` - модуль для обработки JP данных
- `JsonWork.py` - модуль для работы с JSON
- `Search.py` - модуль поиска
- `aifile.txt` - файл с AI-данными
- `dirBook.ico` - иконка директории
- `iconca.ico` - иконка приложения

## Требования
- Python 3.x
- Библиотеки: pandas, openpyxl (предполагаемые, основываясь на использовании Excel)

## Установка
1. Убедитесь, что у вас установлен Python 3.x
2. Установите необходимые зависимости (если есть файл requirements.txt)
3. Запустите основной скрипт в зависимости от ваших потребностей

## Использование
Запустите один из основных модулей в зависимости от типа данных, которые вы хотите обработать:
- `BamProgram.py` для BAM данных
- `CzProgram.py` для CZ данных
- `JpProgram.py` для JP данных
