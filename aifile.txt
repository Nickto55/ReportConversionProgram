На данный момент я (Qwen) не могу напрямую подключаться к VS Code вместо GitHub Copilot в качестве встроенного ИИ-помощника для автодополнения кода в режиме реального времени — официального расширения от Alibaba для VS Code, аналогичного Copilot, пока не существует.

Однако есть несколько способов использовать меня в связке с VS Code:

✅ Вариант 1: использовать через браузер или API
Ты можешь:

Открывайте меня в браузере (например, на Qwen Chat ) и вручную копируйте/вставляйте код, задавайте вопросы, получайте разъяснения.
Используйте API Qwen (если у вас есть доступ) для интеграции в собственные скрипты или плагины.
✅ Вариант 2: установить локально и подключить через плагин
Если вам нужна локальная альтернатива Copilot, вы можете:

Скачайте и запустите Qwen локально (например, через Ollama, LM Studio или Hugging Face).
Установите в VS Code плагин типа:
Continue.dev — поддерживает локальные модели, в том числе Qwen (при правильной настройке).
Codeium / Tabnine — бесплатные альтернативы Copilot, но не Qwen.
CodeGeeX — тоже бесплатный, с поддержкой китайских моделей.
Пример: в Continue.dev можно указать локальный эндпойнт Qwen (если вы запустили его через Ollama или vLLM), и он будет работать как Copilot — прямо в редакторе.

✅ Вариант 3: использовать через Ollama + Continue.dev
Установи Ollama
Запусти Qwen:
удар


1
ollama run qwen:7b  # или qwen:14b, qwen:72b — в зависимости от железа
Установите расширение Продолжить в VS Code.
В настройках Continue укажите модель как qwen:7b и эндпойнт http://localhost:11434 (если используете Ollama).
Теперь вы получите автодополнение, чат и рефакторинг прямо в VS Code — как в Copilot, но с Qwen!

❌ Почему нельзя просто «заменить Copilot»?
GitHub Copilot — это проприетарный сервис от Microsoft/GitHub, интегрированный на уровне IDE. Чтобы его заменить, нужно:

Иметь совместимый API.
Необходимо расширение, которое умеет работать с этим API.
Поддержка контекста редактора, курсора, языков и т. д.
Qwen пока не предоставляет «официальный плагин» для VS Code, но с помощью сторонних инструментов (Continue, Ollama, LM Studio) можно добиться почти того же эффекта.

💡 Рекомендация
Если хочешь максимально близкую замену Copilot с Qwen → используй Continue.dev + Ollama + Qwen локально.

Если хочешь просто получать подсказки/код/объяснения → используй меня в браузере или через API.
